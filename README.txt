Shahjada-Al-Habib-Website
-------------------------

Folder structure:
- index.html
- poems.html
- stories.html
- reviews.html
- about.html
- contact.html
- css/style.css
- js/script.js
- assets/ (place your images here)

Assets (place your images with these filenames in the assets/ folder):
- hero.jpg                 <-- Hero/banner image (recommended 1600x900)
- ShahajadaProfile800800.jpg  <-- Profile portrait (800x800)
- cover1.jpg               <-- Book cover (recommended 600x900)
- thumb1.jpg               <-- Gallery thumbnail (800x600)
- gallery1.jpg             <-- Larger gallery image (1200x800)
You may add more images and update HTML accordingly.

How to use:
1. Download the zip and extract the folder.
2. Open the folder in VS Code or any code editor.
3. Open index.html in Live Server (or any static server) to view the site.
4. Replace placeholder text and images with your real content.
